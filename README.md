# YouTube_clone_project

A responsive YouTube homepage clone built using HTML and CSS.

## Features

- Responsive video grid layout
- Fixed navigation header
- Sidebar navigation menu
- Search bar with icons
- Notification badge
- Video thumbnails with duration labels
- YouTube-inspired design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid

## Preview

This project recreates the appearance of the YouTube homepage for learning and practice purposes.
